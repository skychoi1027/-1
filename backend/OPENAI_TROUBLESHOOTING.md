# OpenAI API 429 오류 해결 방법

## 문제 상황
- 예산: $0.00 / $5 (사용하지 않음)
- 오류: 429 "insufficient_quota"

## 가능한 원인 및 해결 방법

### 1. 예산이 활성화되지 않음
- OpenAI 대시보드에서 예산이 실제로 활성화되었는지 확인
- 예산 설정 후 몇 분 정도 기다려야 적용될 수 있음

### 2. 결제 방법 미등록
- Settings > Billing에서 결제 방법이 등록되어 있는지 확인
- 예산만 설정하고 결제 방법이 없으면 작동하지 않을 수 있음

### 3. API 키가 다른 계정의 것
- 현재 사용 중인 API 키가 예산이 설정된 계정의 것인지 확인
- API 키를 새로 생성하여 테스트

### 4. 무료 크레딧 만료
- 무료 크레딧이 만료되었고, 유료 플랜으로 전환되지 않았을 수 있음
- Usage 페이지에서 크레딧 상태 확인

### 5. 계정 제한
- 새 계정의 경우 일시적인 제한이 있을 수 있음
- OpenAI 지원팀에 문의

## 확인 사항
1. https://platform.openai.com/account/billing 접속
2. 결제 방법 등록 여부 확인
3. 예산 설정이 활성화되어 있는지 확인
4. Usage 페이지에서 실제 사용량 확인
5. API 키가 올바른 계정의 것인지 확인

## 임시 해결책
현재는 기본 조언이 반환되도록 설정되어 있으므로, 서비스는 계속 작동합니다.
OpenAI 계정 문제를 해결하면 실제 AI 조언이 반환됩니다.
